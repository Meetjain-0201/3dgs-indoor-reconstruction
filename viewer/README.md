# viewer/

Three.js web viewer for trained 3D Gaussian splats.

Serves an interactive browser experience: orbit, pan, and fly through the reconstructed indoor scene.

Planned contents:

- `index.html` — viewer entry point.
- `src/` — Three.js scene, splat loader, camera controls.
- `package.json` — frontend dependencies (Three.js, Vite).

Run locally:

```
cd viewer
npm install
npm run dev
```
